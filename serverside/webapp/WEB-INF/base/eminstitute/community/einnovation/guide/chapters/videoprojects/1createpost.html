How to Create Video Project

  1. Navigate to the Video Projects Tab
    Click on the Video Projects tab in the main menu to access the post management area.
  2. Create a Video Project
    Select the Create New button to start a new post.
  3. Navigate to files, upload the file and fill Out the Details
    • Name: Enter the title of your post.
    • Description: Provide a brief description.
    • Date: Add the relevant date.
    • Project: Assign projects if needed for better organization.
  4. Save Your Video
    Once all the details are complete, click Save.
