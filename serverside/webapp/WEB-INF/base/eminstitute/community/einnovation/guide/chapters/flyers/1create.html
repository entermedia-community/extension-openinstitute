7.1. Create a New Flyer 2.1

How to Create a Flyer in eMedia Library

  1. Navigate to the Flyers Tab   
    Click on the Flyerstab in the main menu to access the flyer management area.
  2. Create a New Flyer   
    Select the Create Flyerbutton to start a new flyer.
  3. Fill Out the Details
  • Name: Enter the title of your flyer.
  • Description: Provide a brief description.
  • Date: Add the relevant date.
  • Project: Assign a project for better organization.
  1. Save Your Flyer   
    Once all the details are complete, click Save.
