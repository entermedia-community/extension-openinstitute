Install Lucid link on the server

We setup an eMedia Hot Folder named uservideo in our website. Then in our Docker, we setup Lucid CLI to monitor the Lucid Link files and index them to our server:

lucid3 daemon --fs einnovation.emedia --user tech@emedialibrary.com --root-point uservideo --mount-point /mnt/data/msites/innovation/services/uservideo
