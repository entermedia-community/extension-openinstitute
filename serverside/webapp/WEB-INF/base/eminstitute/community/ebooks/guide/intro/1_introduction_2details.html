Book Details 1.2

##Discuss all the fields? The only required detail is the Working Title. This field is used to find the book as you work on it. so that your book has all the needed data to be shared with others.

Chapters

A chapter will be loaded based on its chapter number or "ordering" field.

Topics

Once a chapter is created you can create Topics. A Topic can be any concept or thought. These can be sorted and selected later.

Questions

Questions are multiple choices that can be used to help the user understand the material
